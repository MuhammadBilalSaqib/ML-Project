# Logistic Regression Classification (Tabular + Image) with Flask

## Description
This project implements Logistic Regression for binary classification on two different datasets: Breast Cancer (tabular data) and Chest X-ray Pneumonia (image data). Both models are trained, evaluated, and deployed using a Flask web application that allows real-time predictions through a browser interface.

## Features
- Logistic Regression implementation
- Tabular dataset classification (Breast Cancer)
- Image dataset classification (Chest X-ray Pneumonia)
- Data preprocessing and feature scaling
- Model evaluation (Accuracy, Precision, Recall, F1-score)
- Confusion matrix display
- Flask web interface
- Real-time predictions

## Technologies Used
- Python
- Scikit-learn
- Flask
- Pandas & NumPy
- Matplotlib & Seaborn
- Pillow (PIL)
- HTML, CSS, JavaScript
